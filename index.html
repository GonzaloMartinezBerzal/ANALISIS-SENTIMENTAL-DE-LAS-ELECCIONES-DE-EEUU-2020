Análisis Sentimental de las Elecciones de EEUU

Objetivo

El objetivo de este proyecto es obtener un análisis sentimental sobre un dataset de tweets generados entre el 15 de octubre de 2020 y el 8 de noviembre de 2020, los cuales incluyen los hashtags #donaldtrump y #joebiden

Este proyecto ha sido realizado dentro de la asignatura Cloud y Big Data de la Universidad Complutense de Madrid por Gonzalo Martínez Berzal, Álvaro Penalva Alberca, Mario Román Dono y Javier Ignacio Sotelino Barriga

Necesidad de Big Data

Actualmente Twitter se ha convertido en un espacio donde un alto número de gente expresa sus opiniones acerca de todo tipo de temas, en especial la política. Por ello, es posible realizar análisis sentimentales para determinar tendencias políticas según el territorio. Además, debido al alto volumen de tweets generados, es imprescindible aplicar técnicas propias de big data para filtrar, traducir, valorar y clasificar los datos de nuestro proyecto.

Descripción de la solución

Utilizando Pandas y TextBlob, un módulo de análisis de lenguaje natural, hemos procesado nuestro dataset para obtener las tendencias por continentes, países y por estados de Estados Unidos. En este último caso, también hemos comparado si el candidato con una mejor valoración de sus tweets relacionados se corresponde con el que resultó ganador, para valorar si el análisis se corresponde con la realidad o no.

Descripción del dataset

El dataset utilizado consta de casi dos millones de tweets recogidos mediante la API de Twitter entre el 15 de octubre de 2020 y el 8 de noviembre de 2020, buscando aquellos que contenían los hashtags #donaldtrump y #joebiden. Se incluye un gran número de información para cada tweet, pero en el ámbito de este proyecto nos hemos centrado únicamente en el texto de los tweets y en la información de localización.

Descripción técnica de la aplicación

Para llevar a cabo nuestro estudio, hemos utilizado Python 3.5, Pandas, una herramienta de Big Data para poder manejar y filtrar nuestro volumen de tweets, TextBlob para realizar el análisis sentimental y Pandarallel para paralelizar nuestra aplicación y obtener un mayor rendimiento. Se incluyen tres scripts paralelizados para el análisis por continentes, por países y por estados de Estados Unidos.

Rendimiento

Gracias al uso de Pandarallel, hemos logrado reducir el tiempo de procesamiento al paralelizar las tareas entre los distintos núcleos del ordenador. Se ha obtenido un decrecimiento del tiempo de 2 minutos y 22 segundos al pasar de usar un solo worker a usar diez workers.

Funciones avanzadas

Debemos destacar el uso de tecnologías avanzadas como el análisis de lenguaje natural en nuestro proyecto o la utilización de herramientas nuevas como Pandas y Pandarallel. Intentamos implementar la paralelización en clúster mediante Dask y Amazon EMR, pero no fue posible debido a problemas con TextBlob.

Conclusiones

En este proyecto hemos aprendido cómo se pueden aplicar técnicas de Big Data para procesar grandes volúmenes de datos y obtener conclusiones relevantes aplicables al mundo real. Además, hemos logrado cumplir el objetivo de nuestro estudio, que era comprobar que el análisis sentimental de tweets es una técnica válida en el ámbito político. Existen varios aspectos que podrían ser modificados de cara a una posible ampliación en el futuro.
