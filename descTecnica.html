Análisis Sentimental de las Elecciones de EEUU

Descripción técnica de la aplicación

Twitter se ha convertido en la última década en uno de los mayores foros de opinión del mundo, ya que cualquier persona, desde el ciudadano de a pie hasta las personas más influyentes, puede expresar su punto de vista sobre todo tipo de temas, siendo la política uno de los más relevantes. Teniendo en cuenta esta realidad, resulta evidente que analizar las tendencias de los tweets según el territorio puede ser de interés para los equipos de campaña de los candidatos políticos para, de esta forma, modelar su discurso y obtener más votos. Con este objetivo, hemos trabajado con un dataset de tweets relacionados con las elecciones de Estados Unidos de 2020, recogidos entre el 15 de octubre de 2020 y el 8 de noviembre de 2020, los cuales contienen los hashtags #donaldtrump y #joebiden.

La herramienta principal que hemos utilizado para llevar a cabo nuestro análisis es Pandas. Esto se debe a la necesidad que existe en nuestro proyecto de utilizar Python 3.5 en adelante. En general, las tecnologías que utilizamos son más óptimas y eficientes con esta versión de Python. Además, con Pandas solventamos los errores de lectura al crear Data Frames con Spark, por ejemplo, Spark confunde saltos de línea, caracteres extraños, emojis y diferentes anomalías que son impredecibles en un posible tweet. Para efectuar el análisis sentimental del texto de los tweets, hemos empleado TextBlob, una librería de Python basada en el Natural Language Toolkit (NLTK) que provee una API sencilla para efectuar estos análisis. En cuanto a la paralelización, hemos utilizado Pandarallel, una librería de Python que permite aumentar la velocidad del procesamiento al paralelizar las tareas entre los núcleos del procesador. Para la traducción de tweets consideramos utilizar la librería GoogleTrans, que utiliza Google Cloud para efectuar las traducciones. No obstante, las consultas gratuitas a esta API son limitadas, por lo que tuvimos que descartar la opción de traducir los tweets como ya hemos explicado. Todos los scripts y los datasets utilizados se pueden consultar en el siguiente repositorio de Github: https://github.com/WantonFrito/bigdata2020-2021ELECTIONS. Dentro del repositorio, existen dos ficheros .csv: trump.csv, que incluye todos los tweets con el hashtag #donaldtrump, y biden.csv, que incluye todos los tweets con el hashtag #joebiden. Se incluyen tres scripts para el análisis por territorios: analisisPorContinentes.py: Analiza los tweets por continente del candidato especificado e imprime por pantalla cuál es su polaridad. Si se incluye la opción de ambos, también imprime cuál de los dos es el preferido en cada continente. Uso: python analisisPorContinentes.py analisisPorPaises.py: Analiza los tweets del país especificado e imprime por pantalla la polaridad de ambos candidatos. Para especificar el país, se debe utilizar el código del país según la norma ISO 3166-1 alpha-3. Si se quiere conocer el resultado de todos los países, se debe utilizar “ALL”. Uso: python analisisPorPaises.py analisisPorEstados.py: Analiza los tweets de cada estado de EEUU, imprime por pantalla la polaridad de ambos candidatos y si el candidato con mayor positividad se corresponde con el ganador de las elecciones en ese estado. Finalmente imprime cuántas coincidencias ha habido y el porcentaje de acierto. Uso: python analisisPorEstados.py Para ejecutar estos scripts, se deben tener instalados los siguientes paquetes: Python >= 3.5 y Pip 3 sudo apt update && sudo apt install python3-pip Pandas pip3 install pandas TextBlob pip3 install textblob && python -m textblob.download_corpora Pandarallel pip3 install pandarallel
